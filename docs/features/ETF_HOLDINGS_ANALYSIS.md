## 1. 핵심 질문

### 1.1 아이디어

```
"반도체 ETF 100개 중 70개가 삼성전자 비중을 늘렸다"
"AI 관련 ETF 80%가 반도체 섹터 비중을 확대했다"

→ 이런 집단적 판단을 측정할 수 있다면 가치 있는 선행 지표가 될 수 있지 않을까?
```

### 1.2 결론

**조건부로 가능하다.**

단, 핵심 조건:
- **액티브 ETF만** 의미 있는 신호를 제공함
- 패시브 ETF의 비중 변화는 단순히 가격 변화의 결과일 뿐
- 데이터 확보에 제약이 있음

## 2. 패시브 vs 액티브 ETF 구분

### 2.1 패시브 ETF (대다수)

```
예: SPY, QQQ, EWY, SOXX 등

작동 방식:
• 지수(Index)를 그대로 추종
• 시가총액 가중 → 주가 상승 시 비중 자동 증가
• 펀드 매니저의 "판단"이 개입하지 않음

삼성전자 비중 20% → 24% 증가 시:
• 삼성전자 주가가 올랐기 때문
• 펀드 매니저가 "삼성전자가 좋다"고 판단한 것이 아님
• 정보 가치: 없음 ❌

결론: 패시브 ETF 비중 변화는 후행 지표
```

### 2.2 액티브 ETF

```
예: ARKK, ARKW, ARKG, MKOR 등

작동 방식:
• 펀드 매니저가 종목과 비중을 직접 결정
• 리서치와 판단에 기반한 의도적 선택
• 매수/매도가 펀드 매니저의 확신을 반영

ARKK가 테슬라 비중 15% → 10%로 축소 시:
• Cathie Wood가 의도적으로 판단한 결과
• "테슬라에 대한 확신이 줄었다" 또는 "다른 기회가 더 좋다"
• 정보 가치: 있음 ✅

결론: 액티브 ETF 비중 변화는 잠재적 선행 지표
```

### 2.3 정보 가치 비교

| 구분 | 비중 변화 의미 | 정보 가치 | 분석 대상 |
|------|---------------|----------|----------|
| 패시브 ETF | 가격 변화의 결과 | 낮음 | ❌ 제외 |
| 액티브 ETF | 매니저의 의도적 판단 | 높음 | ✅ 포함 |
| 섹터 ETF Flow | 투자자들의 자금 이동 | 중간 | ✅ 포함 |

## 3. 데이터 가용성

### 3.1 무료로 얻을 수 있는 데이터

| 데이터 | 소스 | 갱신 주기 |
|--------|------|----------|
| 특정 ETF 현재 보유 종목 | etf.com, etfdb.com, Yahoo Finance | 일간 |
| 특정 종목 포함 ETF 역검색 | etf.com Stock Finder | 일간 |
| ARK ETF 일간 변화 | cathiesark.com, arktrack.com | 일간 |
| ETF Fund Flow | etf.com, etfdb.com | 일간 |

### 3.2 제한적/유료 데이터

| 데이터 | 소스 | 비용 |
|--------|------|------|
| 역사적 비중 변화 | Bloomberg, Refinitiv | 고비용 |
| ETF Holdings API | Finnworlds, Intrinio, Twelve Data | 월 $50~500 |
| 13F 기관 보유 데이터 | SEC EDGAR (무료), WhaleWisdom (유료) | 무료~유료 |

### 3.3 핵심 제약

```
문제: 역사적 비중 변화 데이터 확보가 어려움

• 대부분 플랫폼은 "현재" holdings만 제공
• 과거 비중 데이터는 Bloomberg/Refinitiv 급 필요
• 일부 API도 historical weights를 제공하지 않음

해결책:
• 직접 일간 데이터 수집 → 자체 히스토리 구축
• ARK ETF는 무료로 일간 데이터 확보 가능
```

## 4. 실현 가능한 분석 방법

### 4.1 1단계: ARK ETF 추적 (즉시 실행 가능)

```
대상:
• ARKK (Innovation)
• ARKW (Next Gen Internet)
• ARKG (Genomic Revolution)
• ARKF (Fintech Innovation)
• ARKQ (Autonomous Tech & Robotics)
• ARKX (Space Exploration)

데이터 소스:
• cathiesark.com - 무료, 일간 업데이트
• arktrack.com - 무료, 일간 업데이트

분석 가능한 것:
• 개별 종목 비중 변화 추적
• 신규 편입/제외 종목 감지
• 섹터별 비중 변화 집계
• 매수/매도 패턴 분석

예시 신호:
"ARK ETF 6개 중 5개가 지난 주 반도체 비중 확대"
"ARKK가 테슬라를 5일 연속 매도 중"
```

### 4.2 2단계: 섹터 ETF Flow 분석

```
직접 비중 대신 "자금 흐름"으로 대체

모니터링 대상:
┌─────────────────────────────────────────────────────────────────┐
│  섹터               │  대표 ETF        │  해석                  │
├─────────────────────────────────────────────────────────────────┤
│  반도체             │  SOXX, SMH       │  AI/Tech 투자 심리     │
│  AI/로보틱스        │  BOTZ, ROBO      │  자동화 테마 관심      │
│  클린에너지         │  ICLN, TAN       │  에너지 전환 베팅      │
│  바이오             │  XBI, IBB        │  헬스케어 섹터 심리    │
│  금융               │  XLF, KRE        │  금리/경기 기대        │
│  한국               │  EWY             │  한국 시장 관심        │
│  신흥국             │  EEM, VWO        │  EM 자금 흐름          │
└─────────────────────────────────────────────────────────────────┘

분석 가능한 것:
• 섹터별 자금 유입/유출 추세
• 섹터 간 상대적 선호도 변화
• 급격한 Flow 변화 감지

예시 신호:
"반도체 ETF에 이번 주 $2B 유입 (역대 최고)"
"바이오 ETF 3주 연속 유출, 총 $500M"
```

### 4.3 3단계: 확장 가능성

```
향후 확장 가능:
• 유료 API를 통해 더 많은 액티브 ETF 추적
• 글로벌 액티브 ETF (유럽, 아시아) 포함
• 13F 데이터와 연계한 기관 투자자 분석

현재는 1-2단계에 집중
```

```
│  ─────────────────────────────────                               │
```

```
│  2. 집계 신호 생성                                               │
│  ─────────────────────────────────                               │
│  종목 수준:                                                      │
│  • "종목 X: ARK ETF 6개 중 4개가 비중 증가"                      │

│                                                                  │
│  섹터 수준:                                                      │
│  • "반도체: ARK ETF 평균 비중 15% → 18%"                         │

│  ─────────────────────────────────                               │
```


### 5.4 검증 계획

```
신호의 선행성 검증:
─────────────────────────────────────────────────────────────────

1. 데이터 축적 (최소 6개월)
   • ARK ETF 일간 holdings
   • 섹터 ETF Flow

2. 백테스팅
   • "ARK가 비중 늘린 종목" 이후 성과
   • "ARK가 비중 줄인 종목" 이후 성과
   • 대조군: 시장 평균

3. 신호 유효성 평가
   • 선행 기간: 며칠/몇 주?
   • 정확도: Hit Rate
   • 수익성: 신호 기반 전략 성과

4. 한계 인식
   • ARK = 성장주 집중, 특정 스타일 편향
   • 샘플 수 적음 (ETF 6개)
   • 과적합 위험
```
