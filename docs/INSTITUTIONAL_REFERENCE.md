## 2. JP Morgan Wealth Management (제공된 J.docx)

### 5단계 버블 평가 프레임워크
```
1. 패러다임 전환 → 새로운 기술/산업
2. 신용 가용성 → 자금 조달 용이성
3. 레버리지/밸류에이션 괴리 → 가격 vs 펀더멘털
4. 투기적 피드백 루프 → 모멘텀 자기강화
5. 붕괴 → 트리거 이벤트
```

### 역사적 비교
| 시대 | 대상 | 현재 AI 비교 |
|------|------|-------------|
| 1840s | 철도 | 인프라 과잉 투자 |
| 1990s | 인터넷 | 밸류에이션 광기 |
| 2020s | AI | 데이터센터 1.6% 공실률 (차별점) |

### 핵심 아이디어
- "AI 버블 vs 닷컴: 부채 비율, 레버리지 지표로 구분"
- "가치사슬 단계별 분석: 물리 인프라 → 디지털 인프라 → 플랫폼 → 애플리케이션"

## 3. Goldman Sachs Commodities (골드만삭스.docx)

### 상품 통제 사이클 (4단계)
```
1. 절연 (Insulation) → 국내 공급 확보
2. 확장 (Expansion) → 해외 투자
3. 집중 (Concentration) → 시장 지배력
4. 레버리지 (Leverage) → 가격 통제력
```

### 핵심 방법론
| 상품 | 분석 방법 | 인사이트 |
|------|----------|----------|
| **금** | Conviction Buyer 모델 | 중앙은행 보유량, EMG 구조적 수요 |
| **구리** | 전략적 비축 분석 | 전력망/데이터센터 필수재 |
| **LNG** | 저장 혼잡 모델 | Henry Hub vs TTF 스프레드 |

### 핵심 아이디어
- "금 가격 = ETF 리밸런싱 민감도 + 중앙은행 수요"
- "에너지 공급 파동 이론: FID 데이터 → 2025-2030 공급 타임라인"
- "AI 전력 수요: 유효 예비 용량 지표, 버지니아 데이터센터 집중도"

## 4. Goldman Sachs Market Outlook (골드만삭스의 2026년.docx)

### Gap-Bridging 프레임워크
```
시장 내재 기대 vs 정량적 예측 Gap 분석

예시:
- GDP 예측: 2.5%
- 시장 내재: 1.7%
- Gap: +0.8%p → 매수 기회?
```

### AI 버블 진단 이중 검증
1. **DCF 분석**: 현금흐름 기반 가치 평가
2. **매크로 불균형**: 부채 비율, 레버리지 지표

### 통화 분석
| 유형 | 특성 | 예시 |
|------|------|------|
| High-Beta | 매크로 민감 | AUD, NZD |
| Low-Beta | 안정적 | CHF, JPY |

### 핵심 아이디어
- "인플레이션 분해: 일시적 관세 vs 구조적 AI/중국 디플레"
- "EM 주식 세분화: 기술 민감(한국, 대만) vs 내수(인도, 브라질)"

## 6. 공통 패턴 (Cross-Cutting Themes)

### 6.1 정량적 검증 필수
- 모든 정성적 서사는 데이터로 뒷받침
- 회귀분석, DCF, 시나리오 백테스트

### 6.2 구조적 vs 순환적 분리
| 유형 | 예시 | 투자 함의 |
|------|------|----------|
| 순환적 | 재고 사이클, 금리 사이클 | 타이밍 중요 |
| 구조적 | AI 전환, 탈세계화 | 장기 포지션 |

### 6.3 가치사슬 매핑
```
원자재 → 부품 → 시스템 → 플랫폼 → 애플리케이션 → 최종 사용자
         ↑
      가치 축적 지점 파악
```

### 6.4 리스크 프리미엄 정량화
- 정책 불확실성 → Dot Plot 분산
- 소송 리스크 → 확률적 시나리오
- 지정학 → 공급망 비용 증가율

### 6.5 대안 데이터 활용
- 고용 이민 유입량
- 중앙은행 금 보유량
- 데이터센터 가동률
- 위성 이미지 (재고, 교통량)
