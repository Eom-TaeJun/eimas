# EIMAS 프로젝트 - 여신심사 직무 포트폴리오

> 거시경제 분석 및 리스크 관리 시스템 개발 경험

---

## 1. 프로젝트 개요

**EIMAS (Economic Intelligence Multi-Agent System)**는 거시경제 변수와 시장 데이터를 통합 분석하여 **리스크 점수화** 및 **투자 의사결정을 자동화**하는 시스템입니다.

### 핵심 역량 요약

| 역량 | 내용 |
|------|------|
| **거시경제 분석** | Fed 금리, 국채 수익률, 유동성 지표 실시간 모니터링 |
| **리스크 정량화** | 다차원 리스크 스코어링 (0-100점) |
| **시나리오 분석** | 금리/환율/경기 변수별 시나리오 시뮬레이션 |
| **신용 스프레드 분석** | HY OAS, 회사채-국채 스프레드 모니터링 |
| **포트폴리오 리스크** | VaR, 변동성 예측, 상관관계 분석 |

## 3. 리스크 스코어링 시스템

### 3.1 다차원 리스크 점수 산출

**리스크 스코어 구조:**
```
Final Risk Score = Base Risk + Microstructure Adj. + Bubble Adj.

현재 점수: 10.8/100 (매우 낮음)
- Base Risk (CriticalPath): 10.8
- Microstructure Adjustment: 0.0
- Bubble Risk Adjustment: 0.0
```

### 3.2 리스크 구성요소

**1) 시장 미세구조 리스크 (Microstructure)**
| 지표 | 값 | 해석 |
|------|-----|------|
| 평균 유동성 점수 | 65.2/100 | 양호 |
| VPIN (독성 주문) | < 50% | 정상 |
| Kyle's Lambda | 0.00 | 낮은 시장 충격 |

**2) 버블 리스크 (Greenwood-Shleifer 기준)**
| 기준 | 현재 상태 | 임계값 |
|------|----------|--------|
| 2년 누적 수익률 | 정상 | > 100% 시 경보 |
| 변동성 급증 | 없음 | Z-score > 2 |
| 주식 발행 증가 | 없음 | 유의미 증가 시 |

### 3.3 리스크 등급 체계

| 점수 | 등급 | 여신 권고 |
|------|------|----------|
| 0-20 | 매우 낮음 | 적극적 대출 가능 |

| 40-60 | 중간 | 선별적 접근 |

## 4. 시나리오 분석

### 4.1 금리 시나리오

**Base Case (55% 확률)**
- Fed 금리 동결 유지
- 10Y 국채 3.8-4.5% 박스권
- 대출 금리: 안정적

**Bull Case (30% 확률)**

- 인플레이션 안정화
- 대출 수요 증가 예상

**Bear Case (15% 확률)**

- Fed 긴축 재개
- 신용 스프레드 확대 → 부실 위험 증가

## 5. 포트폴리오 리스크 관리

### 5.1 변동성 예측 (GARCH 모델)

**GARCH(1,1) 결과:**
```
현재 변동성: 19.4%
10일 예측 변동성: 14.3%
Persistence (α+β): 0.897
Half-life: 6.3일
```

**해석:**
- 변동성 클러스터링 존재 (충격 지속성 89.7%)
- 현재 변동성 → 평균 회귀 중 (감소 추세)

### 5.2 자산 상관관계 분석

**DTW (Dynamic Time Warping) 결과:**
```
가장 유사: SPY ↔ QQQ (상관 높음)
가장 상이: VIX ↔ UUP (분산 효과)
Lead-Lag: SPY가 QQQ보다 1일 선행
```

**Systemic Similarity (D̄ matrix):**
- 자산 간 시스템적 연관성 정량화
- 분산투자 효과 측정 가능

### 5.3 포트폴리오 최적화 (HRP)

**Hierarchical Risk Parity 결과:**
```
방법론: MST(Minimum Spanning Tree) + HRP
장점: 공분산 역행렬 불안정성 해결
적용: 대규모 자산(100개+) 처리 가능
```

| 자산 | 최적 비중 | 역할 |
|------|----------|------|
| SPY | 37.1% | 핵심 자산 |
| QQQ | 33.5% | 성장 노출 |
| GLD | 24.7% | 인플레 헤지 |
| TLT | 4.7% | 금리 민감 |
